Introduction to HTML.

What is HTML?

HTML stands for Hyper Text Markup Language

HTML is the foundation of every webpage. It tells the browser what content to show and how it's structured.


HTML is used to:

  • Add text (headings, paragraphs)
  • Insert images
  • Create links
  • Organize content into sections, lists, or tables
  • Add forms for user input

HTML Tags

  • <html> - Root element of a webpage
  • <head> - Contains metadata and links
  • <title> - Sets the page title
  • <body> - Contains visible page content
  • <h1> to <h6> - Headings, largest to smallest
  • <p> - Paragraph
  • <a> - Anchor for hyperlinks
  • <img> - Displays an image
  • <ul>, <ol>, <li> - Lists
  • <div> - Generic container for content

Example:

<h1>Welcome to My Website</h1>
                  <p>This is a simple webpage using HTML.</p>
                  <img src="image.jpg" alt="A sample image">

Introduction to CSS

What is CSS ?

CSS stands for Cascading Style Sheets

CSS controls the appearance of your website. It makes your website look Exquisite and professional.

Think of it like: The design, paint, and layout that goes over the HTML body.


CSS is used to:

  • Set colors and fonts
  • Change layouts (like grids or flexboxes)
  • Add spacing with margin/padding
  • Add borders, shadows, hover effects
  • Make the page responsive to different devices

CSS Properties

  • color - Text color
  • background-color - Background color
  • font-family - Typeface of the text
  • font-size - Size of the text
  • margin - Space outside elements
  • padding - Space inside elements
  • border - Border around elements
  • box-shadow - Shadow effect

Example:


        body {
            background-color: #f0f0f0;
            font-family: 'Arial', sans-serif;}
            h1 {
                color: #3498db;
                text-align: center;}
